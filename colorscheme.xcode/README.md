# Xcode 4 themes for Vim

A GUI only port of the Xcode 4 themes *Midnight* and *Dusk* with a dark
background and pleasant colors.

## Installation

Just copy `xcode-midnight.vim` and `xcode-dusk.vim` to `~/vim/colors`.
